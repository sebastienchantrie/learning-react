## Ex06 - Subject

State & Liste

### Sujet

```
Faite en sorte que le composant `Ex06` affiche sa propriété `data` qui est un tableau de nombre.
Il affichera cette liste à l'aide de ul et li.
Créer un bouton: "générer", qui va ajouter un nombre à la liste de donnée. Le nombre ajouté est le nombre de click qu'il y a eu sur le bouton.
Lors d'un clique sur une element, il sera supprimé des données.
```
